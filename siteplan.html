Oregon Camp & Tourist Album

Website Plan

Site Name

Oregon State Park and Tourists Album

Site Purpose

Introduce some Oregon State Parks and tourist sites, while also using the Filtered Temples Javascript as the sorting template, namely for size and age.

Scenarios

Scenarios are questions asked by a site visitor..

  • What kind of tourist and park locations does Oregon have?
  • When was this park or tourist site founded?
  • Size of the actual location in square feet?
  • Additional info on if a given location is currently closed or under maintenance?

Color Schema

The selected color schema for Oregon State Park Album is:

  • Orange (#FFA500) - Used for headings and accents.
  • Dark Medium Gray (#A9A9A9) - Used for background color.
  • Black (#000000) - Used for overall text color.

Typography

The typography selections for Oregon State Park Album are:

  • Roboto, sans-serif - Used for body text and headings.
